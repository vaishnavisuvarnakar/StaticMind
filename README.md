# StaticMind
A Chrome Extension that intercepts distracting websites with a mindful pause overlay — 20 second timer, then choose to continue or talk to an AI refocus chatbot.

Built with vanilla JS, Manifest V3, and a BYOK AI API. Chatpage is a single HTML file deployed on Vercel. No backend, no frameworks.

Black UI, Turkish Wedding gradient (#B9D1EB → #F876DE), Century Gothic in the extension, Outfit on the chatpage, slow glowing geometric shapes throughout.
Conceptualised and designed by Vaishnavi Suvarnakar. If you build on this, a shoutout goes a long way.
